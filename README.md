# Developers Site

## Installation

```
./setup.sh
```

## Development

To start the build watcher and run a http server on port 8080

```
./run.sh
```

### Shutting down

```
fg
```

And then CTRL-C the process

## Releasing

```
./release.sh
```

Note: Our web fonts only work under hosts matching `*.theguardian.com`.
